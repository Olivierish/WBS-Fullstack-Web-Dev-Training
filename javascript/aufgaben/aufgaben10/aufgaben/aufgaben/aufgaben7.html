Tag 7

Auswahlliste / DOM

Erstelle ein Array. Die Elemente des Arrays sind Objekte.

Die Objekte enthalten Filme. Filmtitel, Erscheinungsjahr, Genre, Regisseur, Hauptdrasteller

Die Hauptdarsteller sind wiederum ein Array.


Erstelle händisch eine Auswahliste als HTML select-Element.
Füge programmatisch die Filmtitel aus Auswahlfelder der Auswahlliste hinzu.

Wenn der User auf ein eines der Filmtitel im Select Feld aufruft. Soll der Film mit allen Eigenschaften angzeigt werden und zwar in den Spalten einer Tabellenreihe.

Die erste Reihe der Tabelle enthält Kopfzellen mit den Beschreibungen der Objekt Eigenschaften: Titel, Erscheinungsjahr, Genre, Regisseur, Schauspieler

In den Aufgaben befinden sich ein paar Bilder. Eine weitere Herausforderung wäre, auch ein Bild zum Film in der Tabelle anzeigen zu lassen.

Eine weitere Herausforderung wäre, eine Mehrfachauswahl zuzulassen. So dass man sich auch mehrere Filme gleichzeitig anzeigen lassen kann.
