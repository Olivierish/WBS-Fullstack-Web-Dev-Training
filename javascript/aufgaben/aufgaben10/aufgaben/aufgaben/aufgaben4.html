Aufgaben Tag 4

1

Funktion und Event erforderlich

Erstelle ein Array mit Tieren, genannt afrikaTiere[]. Erstelle ein Array zirkusTiere[] in dem sich noch keine Elemente befinden. Erstelle eine Funktion und einen Button, welcher die Funktion aufruft. In der Funktion wird eine Zufallszahl erzeugt, welche sich im Bereich der Anzahl der tiereAfrikas befindet.

Anhand der Zufallszahl wird ein Tier aus tiereAfrikas gelöscht und im Array zirkusTiere am Anfang eingefügt.

2

Erzeuge ein Programm welches auf Buttonclick, die 4er Reihe anzeigt ( Das Einmal Vier). Das sollte aber programmatisch erstellt werden, anstatt einfach hingeschrieben sein.

3

Erzeuge mehrere Buttons, die alle die gleiche Funktion aufrufen. In der Funktion, wird das einmal eins, einmal zwei, einmal 3 etc. erzeugt und auf der Seite angezeigt.

Welches einmal eins es ist, wird über ein Argument definiert.

4

Erzeuge mehrere Arrays mit Elementen deiner Wahl.
Auf Buttonklick sollen die Arrayelemente der Reihe nach im p Element jeweils einzeln mit Zeilenumbruch angezeigt werden. Es sollte dort nicht undefined angezeigt werden.

Welches Array wird über ein Argument definiert.

5

Erzeuge im HTML Markup ein Bild in einem Figure Element.

Erzeuge ein paar Buttons, die eine Funktion aufrufen.

Die Funktion soll bei jedem Klick ein anderes Bild anzeigen, welches Bild es ist wird über ein Argument bestimmt.

5a

Erzeuge im HTML Markup ein Bild in einem Figure Element.

Erzeuge einen Button, der eine Funktion aufruft.

Die Funktion soll bei jedem Klick ein anderes Bild aufrufen.

Benutze dazu Bilder die namentlich durchnummeriert sind. 1_bild.jpg, 2_bild.jpg

oder 1.jpg, 2.jpg, 3.jpg.... Hauptsache sie haben alle den gleichen Namen mit einer durchlaufenden Nummer.

Wenn das Ende der vorhandenen Bilder erreicht ist, sollte wieder das erste Bild angzeigt werden.

 

5b

Erzeuge im HTML Markup ein Bild in einem Figure Element.

Erzeuge zwei Buttons, die weitere Bilder aufrufen und anzeigen.

Ein Button, läuft die Bilderreihe vorwärts, der andere die Bilderreihe rückwärts.

Du kannst es so programmieren, das am Ende angekommen keine weiteren Bilder mehr angzeigt werden und am Anfang angekommen genauso.

Wenn du es schaffst, es so zu programmieren, dass beim Ende wieder das erste Bild angzeigt wird, wenn man am Anfang angekommen ist, das letzte Bild angezeigt wird, kannst du auch das programmieren.

6

Zur der Bilderliste des heutigen Tages, erzeuge ein Array mit Bildtiteln.

Erzeuge eine Funktion, die durch einen Button aufgerufen wird.

Die Funktion zeigt ein zufälliges Bild an, und dazu den dazugehörigen Bildtitel, im figcaption Element.

Achte darauf, wieviele Bilder zur Verfügung stehen, damit du keine Zufallszahl erzeugst, die nicht funktioniert.

7

Erstelle ein Array in dem Mitarbeiter aufgeführt sind mit Namen, und Beruf. Es sollte kein mehrdimensionales Array sein.

Erstelle eine Funktion ohne Argumente.

Beim Aufruf der Funktion sollten aus dem Array einige Mitarbeiter/ Arrayelemente angezeigt werden.

Die Anzahl der angezeigten Elemten variiert jedoch je nach Funktionsaufruf.

Beim Aufruf wird über Parameter definiert, welche Mitarbeiter angezeigt werden.

Das kann bei einem Aufruf mal Mitarbeiter mit Index 6 sein, oder 3 Mitarbeiter mit den Indicies, 0,4,5

Erstelle ein paar Buttons welche die Funktion aufrufen.

z.B. Wenn die Funktion showTeam heißen würde, würde bei folgendem Funktionsauf, die Mitarbeiter von Index 2, 4 und 6 angezeigt.

showTeam(2,4,6)

Tipp: Funktion arguments oder ...restArgs
