automatisches System zu schreiben, das sie gerade aufbauen. Sie möchten damit herausfinden,
ob jemand einen Film mit Altersfreigabe FSK 12 anschauen darf oder nicht.
Das sind die Regeln: Wer 12 Jahre oder älter ist, darf den Film alleine anschauen. Ist
die Person noch keine 12 Jahre alt, aber in Begleitung eines Sorgeberechtigten, darf sie
ebenfalls ins Kino hinein. Andernfalls darf sie den Film nicht anschauen.
var alter = 11;
var begleitet = true;
???
Stelle das Beispiel fertig, um mithilfe der Variablen alter
und begleitet herauszufinden, ob dieser 11-Jährige den
Film anschauen darf. Versuche auch, andere Werte einzusetzen
(setze zum Beispiel alter auf 12 und begleitet
auf false), und schau, ob dein Code immer noch das
richtige Ergebnis liefert.

Array Aufgaben

1

Erzeuge eine Array mit Wochentagen. Füge das letzte Element an den Anfang. Verwende dazu 2 Array Methoden. Für den Vorgang soll nur eine Zeile benötigt werden.

2

Füge das erste Element des Arrays an das Ende des Arrays. Verwende dazu 2 Array Methoden. Für den Vorgang soll nur eine Zeile benötigt werden.

3

Erzeuge 3 Arrays mit Afrikas Tiere, Haustiere, Nutztiere. Erzeuge ein Array tiere[] und füge alle Elemente der 3 Arrays in dieses Array tiere. Für den Vorgang benötgst du nur eine Zeile. Die Elemente des Arrays Tiere sollten auf der Seite angezeigt werden.

4

Erzeuge ein mehrdimesionales Array person, mit Elementen in denen der Reihe nach Vorname, Name, Ort, Beruf und Geburtsjahr einer Person aufgeführt ist. z.B:
Frieda, Müller, Berlin, LKW-Fahrerin, 1959


Erzeuge einen Textabsatz in dem eins dieser Elemente aufgeführt wird, aber jeweils mit der Bezeichnung davor.für Name, Beruf, Geburtsjahr etc.. Achte auf die nötigen Leerzeichen.

5

Erzeuge ein mehrdimensionales Array mit Filmtitel, Regie, Erscheinungsjahr, FSK Alter

Bei jedem neuen Laden der Seite wird ein zufälliger Film auf der Seite angezeigt mit allen Daten in einzelnen Reihen.
