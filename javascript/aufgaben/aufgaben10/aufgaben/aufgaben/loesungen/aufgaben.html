Zufällige Strings generieren
Baue dir einen Zufallsgenerator für Strings. Zuerst brauchst du dafür einen String mit
allen Buchstaben des Alphabets, allen Zahlen und dem Ausrufezeichen:
var alphabet = "abcdefghijklmnopqrstuvwxyz0123456789!";
Um daraus einen zufälligen Buchstaben herauszupicken, kannst du den Code für
eine Zufallszahl nutzen Math.floor(Math.random()* alphabet.length).
Damit erhältst du einen zufälligen Index innerhalb des Strings. Mit
eckigen Klammern kannst du dann den Buchstaben an dieser Position auslesen.
Um den zufälligen String zu erzeugen, fängst du zuerst mit einem leeren String an
(var zufallString = ""). Dann legst du eine while-Schleife an, die immer neue Buchstaben
an den String anhängt, solange dieser kürzer als 6 Zeichen (oder eine beliebige, von
dir festgelegte Länge) ist. Zum Anfügen eines neuen Buchstabens ans Ende der Zeichenkette
könntest du den Operator += verwenden. Nachdem die Schleife beendet ist,
schreibst du dein Ergebnis in die Konsole, wo du es dann begutachten kannst!
oder fügst es mit innerHTML in ein HTML Element ein
