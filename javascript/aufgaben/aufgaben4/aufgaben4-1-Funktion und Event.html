Funktion und Event erforderlich

Erstelle ein Array mit Tieren, genannt afrikaTiere[]. Erstelle ein Array zirkusTiere[] in dem sich noch keine Elemente befinden. Erstelle eine Funktion und einen Button, welcher die Funktion aufruft. In der Funktion wird eine Zufallszahl erzeugt, welche sich im Bereich der Anzahl der tiereAfrikas befindet.

Anhand der Zufallszahl wird ein Tier aus tiereAfrikas gelöscht und im Array zirkusTiere am Anfang eingefügt.
