5a

Erzeuge im HTML Markup ein Bild in einem Figure Element.

Erzeuge einen Button, der eine Funktion aufruft.

Die Funktion soll bei jedem Klick ein anderes Bild aufrufen.

Benutze dazu Bilder die namentlich durchnummeriert sind. 1_bild.jpg, 2_bild.jpg

oder 1.jpg, 2.jpg, 3.jpg.... Hauptsache sie haben alle den gleichen Namen mit einer durchlaufenden Nummer.

Wenn das Ende der vorhandenen Bilder erreicht ist, sollte wieder das erste Bild angzeigt werden.

 

5b

Erzeuge im HTML Markup ein Bild in einem Figure Element.

Erzeuge zwei Buttons, die weitere Bilder aufrufen und anzeige.

Ein Button, läuft die Bilderreihe vorwärts, der andere die Bilderreihe rückwärts.

Du kannst es so programmieren, das am Ende angekommen keine weiteren Bilder mehr angzeigt werden und am Anfang angekommen genauso.

Wenn du es schaffst, es so zu programmieren, dass beim Ende wieder das erste Bild angzeigt wird, wenn man am Anfang angekommen ist, das letzte Bild angezeigt wird, kannst du auch das programmieren.
