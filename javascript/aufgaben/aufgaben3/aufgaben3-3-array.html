Erzeuge ein Array mit Sprüchen, die als Slogan auf der Seite angezeigt werden können.
Erzeuge ein p Element mit der id="output".
Mit folgender Zeile Code kannst du etwas in dem p Element anzeigen lassen:
document.getElementById("output").innerHTML = "Hallo";
Beim jedem Aufruf der Seite soll ein zufällig ausgewählter Spruch angezeigt werden. Siehe Math Object.
Wenn man das Array um weitere Elemente erweitert sollte am Code für den Zufallsspruch keine Änderung erforderlich sein.
