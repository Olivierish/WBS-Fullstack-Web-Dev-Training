Boxmodelle und Besonderheiten

klassisches Boxmodell

Im visuellen Boxmodell erfolgt die Darstellung durch das zeichnen von Rechtecken. Die Bestandteile dieser Rechtecke beschreibt das Boxmodell

Die Container bestehen aus:

  1. Dem Inhaltsbereich, also der Fläche die durch Texte und oder Eigenschaften wie Breite und Höhe angegeben wird
  2. einem Innenabstand (padding)
  3. einem Rahmen (border)
  4. einem Außenabstand (margin)

Bei Blockelementen können Höhe und Breite beliebig festgelegt werden, bei Inlineelementen werden die Maße durch den Inhalt vorgegeben.
Innen- und Außenabstände können für jede der 4 Seiten einer Box einzeln festgelegt werden.
Wird eine Box positioniert, beginnt die linke Außenkante bei left, die obere Außenkante bei top, die rechte Außenkante bei right und untere Außenkante bei bottom.

Umschließender Container

Innencontainer

.box.prozent{width:50%;height:50%;background:#ccc;}

Höhenangaben in Prozent beziehen sich standarmäßig auf den Inhalt sofern es kein Elternelement gibt welches selbst keine inhaltsabhängige Höhenangabe besitzt!

Margin , Padding und Border am inline-Element

Inlineelemente können standardmäßig keine Höhen , Breiten und vertikale Margins aufnehmen. Vertikale Innenabstände (padding) und Rahmen dehnen ein Inlineelement lediglich visuell (Hintergrund ebene vergrößert sich) aus die Elementhöhe selbst wird damit nicht verändert.

Sonderverhalten von Außenabständen

Zusammenfallende Außenabstände (collapsing margins)

Zwei oder mehr benachbarte Außenabstände (margins) können sich miteinander verbinden und so einen einzigen Abstand bilden. Dieser Vorgang wird zusammenfallen genannt und der so entstandene Abstand wird zusammengefallener Außenabstand (collapsed margin) genannt. Horziontale Außenabstände fallen nicht zusammen.

  • Außenabstand unten ul li{margin-bottom:20px;}
  • Außenabstand oben ul li:last-child{margin-top:40px;margin-bottom:60px;}

Margins von Kinderelementen können sich über Ihr Elternelement ausdehnen. Dies kann verhindert werden wenn dem Elternelement in der vertikalen Rahmen oder Innenabstände zugewiesen werden.
Alternativ verwenden Sie an dem jeweiligen Kinderelement in der vertikalen padding statt margin. Das verwenden alternativer Darstellungseigenschaften wie table oder flex kann dieses Verhalten ebenso unterbinden - ist jedoch oft mit dem Verlust weiterer Standardeigenschaften verbunden.

überlappende Außenabstände

.margin-drop,ul{
/*border-top:1px solid transparent;
border-bottom:1px solid transparent;*/
padding-top:1px;
padding-bottom:1px;
/* display:table; */
}

Horizontales Zentrieren von Blockelementen;

Zum horizontalen zentrieren von Blockelementen selbst muss eine Breite festgelegt werden. Die Zentrierung selbst findet dann mit dem Außenabstand rechts und links sowie dem Wert = auto statt. Der Wert auto sorgt in diesem Fall dafür das der übrige zur Verfügung stehende Platz berechnet und als halbierter Wert der rechten und linken Seite hinzugefügt wird.

inlineelement em.box, p.box{
display:inline-block;
}

Textabsatz em.box, p.box{
display:inline-block;
}

/* Zentrieren von Blockelementen*/
.center{
width:70%;
margin-left:auto;
margin-right:auto;
text-align:center;/* wirkt nur auf inline-inhalte*/
}
